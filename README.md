# Gemini Clone

A simple web-based AI chatbot powered by the Gemini API. Enter a prompt, and get an AI-generated response instantly!

## Features

- Enter any valid prompt to receive an AI-generated response.
- Displays responses dynamically.
- Responsive design with an intuitive user interface.

## How to Use

1. Open the application in your browser.
2. Enter your query in the input field.
3. Click the **Send** button to generate a response.
4. The response will be displayed below the input field.

## Technologies Used

- **HTML:** Structure of the application.
- **CSS:** Styling for a clean and responsive design.
- **JavaScript:** Handles user interactions and dynamic updates.
- **Node.js:** Backend runtime environment.
- **Express.js:** Web framework for API routing.
- **Gemini API:** Generates AI responses based on user queries.

## Installation

```sh
# Clone the repository
git clone https://github.com/yourusername/gemini-clone.git

# Navigate to the project directory
cd gemini-clone
```

## Setup API Key

1. Obtain a Gemini API key from the official Gemini API provider.
2. Add your API key in `script.js`:
   ```js
   const API_KEY = "your-gemini-api-key";
   ```
